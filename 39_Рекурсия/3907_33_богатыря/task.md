# 33 богатыря

**Ограничение времени:** 1 секунда

**Ограничение памяти:** 64Mb

**Ввод:** стандартный ввод или `input.txt`

**Вывод:** стандартный вывод или `output.txt`

Помнится, богатыри в сказке Пушкина так и ходили всегда стройными рядами, отсортированные по неубыванию роста. А что делать, если подрос новый богатырь? Нужно найти ему место в стройных рядах.

Напишите рекурсивную функцию `epic_hero(arr, knight)`, которая вставит элемент в нужное место, чтобы неубывающий порядок сохранился.

Использование рекурсии обязательно.

## Пример 1

**Ввод**
```
heroes = [1, 2, 3, 5, 7]
print(epic_hero(heroes, 0))
```

**Вывод**
```
[0, 1, 2, 3, 5, 7]
```

## Пример 2

**Ввод**
```
heroes = [1, 2, 3, 5, 7]
print(epic_hero(heroes, 4))
```

**Вывод**
```
[1, 2, 3, 4, 5, 7]
```

## Пример 3

**Ввод**
```
heroes = [1, 2, 3, 5, 7]
print(epic_hero(heroes, 8))
```

**Вывод**
```
[1, 2, 3, 5, 7, 8]
```
